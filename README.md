[![Download / Open the app](https://img.shields.io/badge/Open%20the%20app-blue?style=for-the-badge)](https://github.com/friskinggingerol886/bedrock-nbt-editor)

## 🧭 What this is

bedrock-nbt-editor is a browser-based NBT editor for Minecraft Bedrock Edition. It helps you open a `level.dat` file, view its tags, and change values like `LevelName`, game rules, and other NBT data.

It is made for users who want a simple way to edit a Bedrock world file from a web browser. You do not need to install a desktop app first. On Windows, you can use your browser to open the app and work with your world data.

## 🚀 Getting started

Follow these steps to edit a Bedrock world file.

### 1. Find your world folder

Open the folder where Minecraft stores your Bedrock worlds. In many Windows setups, the world files are inside the Minecraft app data area. Find the world you want to change and locate `level.dat`.

### 2. Open the editor

Go to the app through the link above. If you downloaded a local copy, open the app file in your browser. Wait for the editor to load.

### 3. Load `level.dat`

Use the file picker in the editor and select your `level.dat` file. The app reads the NBT data and shows it in a clear tree view.

### 4. Edit the values you want

You can change common values like:

- `LevelName`
- game rules
- player and world tags
- other NBT fields in the file

Click the value you want to edit and enter the new text, number, or setting.

### 5. Save your changes

After you finish, use the save option in the editor. Save the edited file back to your world folder or export it to a safe place first.

### 6. Test the world in Minecraft

Open Minecraft Bedrock and load the world. Check that the world name and settings changed the way you wanted.

This editor lets you see that structure without needing to read raw file data. It is built for people who want a visual way to work with Bedrock world files.

## 🛠️ Troubleshooting

### The file does not open

- Make sure you selected `level.dat`
- Check that the file is from a Bedrock world
- Try opening the page in another browser
- Confirm that the file is not in use by Minecraft

### The editor shows empty data

- Reload the page
- Select the file again
- Make sure the file was copied correctly
- Check that the file is not damaged

### My changes do not appear in Minecraft

- Verify that you saved the edited file
- Replace the world’s `level.dat` with the saved version
- Close Minecraft before testing
- Open the right world folder

### Windows blocks the file

- Move the app files to a normal folder, such as Documents
- Use a browser that can open local files
- Try running the page from the GitHub link again
